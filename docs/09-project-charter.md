# Project Charter

## Working Name

Battery-Aware Grid Intelligence

## Long-Term Mission

Build software that helps BESS owners, EV depots, microgrids, and renewable-heavy grid operators make decisions that respect both financial value and physical asset limits.

## Initial Wedge

PV reliability analytics during the APS phase.

Why this wedge works:

- It is accessible from the current internship path.
- It builds industrial reliability discipline.
- It creates a credible technical artifact using imaging, statistics, and engineering reasoning.
- It links naturally into battery degradation, BESS diagnostics, and grid-edge intelligence.

## First Serious Project

Data-driven reliability assessment of industrial photovoltaic modules using electroluminescence imaging, accelerated aging if available, and statistical modeling.

## Product Ladder

1. EL reliability analytics
2. Battery Health API
3. Degradation Cost Engine
4. Degradation-Aware EMS
5. Grid-Edge EMS
6. Inverter Event Forensics
7. Grid Stability Intelligence

## Near-Term Success Criteria

Within 90 days:

- A clean internship log habit exists.
- A PV reliability research proposal exists.
- A basic EL image feature extraction prototype exists, even if it uses public/sample data.
- A statistical analysis template exists.
- A portfolio narrative connects APS to BESS/grid software.

Within 1 year:

- A credible PV reliability project is complete.
- A BESS health analytics prototype has started.
- The career narrative points toward battery, grid, EMS, or power electronics internships.

## Positioning

Weak framing:

> AI for energy.

Strong framing:

> Physics-informed analytics for battery and grid assets.

Best company thesis:

> We help BESS owners, EV depots, and microgrid operators maximize revenue without destroying battery lifetime or compromising grid reliability.
